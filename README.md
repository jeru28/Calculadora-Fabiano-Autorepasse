# Fabiano Auto Repasse

Calculadora standalone em HTML único, pensada para uso rápido em desktop e mobile.

## Estrutura

- [`index.html`](./index.html): interface, regras de cálculo e validações

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Preencha os dados do veículo e da operação.
3. Revise os alertas antes de copiar o relatório.

## Observações

- A calculadora agora usa apenas a versão standalone.
- Não há mais dependência da implementação React anterior.
